Markdown 是一种轻量级的标记语言，广泛用于编写技术文档和学术文章。其内置的数学公式功能通过 LaTeX 语法支持复杂的方程式编写，为用户高效呈现科学内容。本指南将全面讲解所有 Markdown 数学语法规则、符号及高级用法，适用于在 GitHub、Obsidian、Typora 等平台精准渲染数学公式。

1. 数学公式基本语法

  • 语法: 用单个美元符号包裹公式 $...$

  • 语法: 用双美元符号包裹公式 $$...$$

5. 高级排版技巧

  • 数学字体控制：

    • 粗体: $\mathbf{A}$ → A\mathbf{A}A
    • 空心字体: $\mathbb{R}$ → R\mathbb{R}R (实数集)
    • 手写体: $\mathcal{L}$ → L\mathcal{L}L

  • 多行公式对齐：

    $$
\begin{align}
(a + b)^2 &= a^2 + 2ab + b^2 \\
          &= a^2 + b^2 + 2ab
\end{align}
$$

    效果:

    (a+b)2=a2+2ab+b2=a2+b2+2ab\begin{align} (a + b)^2 &= a^2 + 2ab + b^2 \\ &= a^2 + b^2 + 2ab \end{align} (a+b)2​=a2+2ab+b2=a2+b2+2ab​​

6. 跨平台兼容性参考

工具/平台支持程度
GitHub支持 $$...$$ 块级公式，但需开启 MathJax 渲染
Obsidian需安装插件 "Advanced Math"，支持行内与块级公式
Typora默认支持 KaTeX 渲染，实时预览
Jupyter完美支持 LaTeX 公式，在 Markdown 单元格中直接使用
VS Code需安装插件（如 "Markdown+Math"），支持实时预览
