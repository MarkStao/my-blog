1. 集合论基础

1.1 集合运算公式

  • 并集：A∪B={x∣x∈A∨x∈B}A \cup B = \{x \mid x \in A \lor x \in B\}A∪B={x∣x∈A∨x∈B}
  • 交集：A∩B={x∣x∈A∧x∈B}A \cap B = \{x \mid x \in A \land x \in B\}A∩B={x∣x∈A∧x∈B}
  • 补集：A‾={x∣x∉A}\overline{A} = \{x \mid x \notin A\}A={x∣x∈/A}
  • 差集：A−B={x∣x∈A∧x∉B}A - B = \{x \mid x \in A \land x \notin B\}A−B={x∣x∈A∧x∈/B}
  • 对称差：A⊕B=(A−B)∪(B−A)A \oplus B = (A - B) \cup (B - A)A⊕B=(A−B)∪(B−A)

1.2 集合运算律

  • 交换律：A∪B=B∪AA \cup B = B \cup AA∪B=B∪A，A∩B=B∩AA \cap B = B \cap AA∩B=B∩A
  • 结合律：(A∪B)∪C=A∪(B∪C)(A \cup B) \cup C = A \cup (B \cup C)(A∪B)∪C=A∪(B∪C)，(A∩B)∩C=A∩(B∩C)(A \cap B) \cap C = A \cap (B \cap C)(A∩B)∩C=A∩(B∩C)
  • 分配律：A∪(B∩C)=(A∪B)∩(A∪C)A \cup (B \cap C) = (A \cup B) \cap (A \cup C)A∪(B∩C)=(A∪B)∩(A∪C)，A∩(B∪C)=(A∩B)∪(A∩C)A \cap (B \cup C) = (A \cap B) \cup (A \cap C)A∩(B∪C)=(A∩B)∪(A∩C)
  • 德摩根定律：A∪B‾=A‾∩B‾\overline{A \cup B} = \overline{A} \cap \overline{B}A∪B=A∩B，A∩B‾=A‾∪B‾\overline{A \cap B} = \overline{A} \cup \overline{B}A∩B=A∪B

2. 命题逻辑与谓词逻辑

2.1 命题逻辑公式

  • 蕴含式：p→q≡¬p∨qp \rightarrow q \equiv \neg p \lor qp→q≡¬p∨q
  • 等价式：p↔q≡(p→q)∧(q→p)p \leftrightarrow q \equiv (p \rightarrow q) \land (q \rightarrow p)p↔q≡(p→q)∧(q→p)
  • 逆否命题：p→q≡¬q→¬pp \rightarrow q \equiv \neg q \rightarrow \neg pp→q≡¬q→¬p
  • 常用逻辑等价式：
    • 双重否定律：¬¬p≡p\neg \neg p \equiv p¬¬p≡p
    • 幂等律：p∨p≡pp \lor p \equiv pp∨p≡p，p∧p≡pp \land p \equiv pp∧p≡p
    • 吸收律：p∨(p∧q)≡pp \lor (p \land q) \equiv pp∨(p∧q)≡p，p∧(p∨q)≡pp \land (p \lor q) \equiv pp∧(p∨q)≡p

2.2 谓词逻辑公式
