1. 集合论

1.1 基本概念

  • 集合表示：

    • 列举法：A={1,2,3}A = \{1, 2, 3\}A={1,2,3}
    • 描述法：A={x∣x 是正整数且 x≤3}A = \{x \mid x \text{ 是正整数且 } x \leq 3\}A={x∣x 是正整数且 x≤3}

  • 集合运算：

    • 并集：A∪B={x∣x∈A 或 x∈B}A \cup B = \{x \mid x \in A \text{ 或 } x \in B\}A∪B={x∣x∈A 或 x∈B}
    • 交集：A∩B={x∣x∈A 且 x∈B}A \cap B = \{x \mid x \in A \text{ 且 } x \in B\}A∩B={x∣x∈A 且 x∈B}
    • 差集：A−B={x∣x∈A 且 x∉B}A - B = \{x \mid x \in A \text{ 且 } x \notin B\}A−B={x∣x∈A 且 x∈/B}
    • 补集：A‾={x∣x∉A}\overline{A} = \{x \mid x \notin A\}A={x∣x∈/A}

  • 幂集：

    • P(A)={S∣S⊆A}P(A) = \{S \mid S \subseteq A\}P(A)={S∣S⊆A}

1.2 集合恒等式

  • 交换律：

    • A∪B=B∪AA \cup B = B \cup AA∪B=B∪A
    • A∩B=B∩AA \cap B = B \cap AA∩B=B∩A

  • 结合律：

    • A∪(B∪C)=(A∪B)∪CA \cup (B \cup C) = (A \cup B) \cup CA∪(B∪C)=(A∪B)∪C
    • A∩(B∩C)=(A∩B)∩CA \cap (B \cap C) = (A \cap B) \cap CA∩(B∩C)=(A∩B)∩C

  • 分配律：

    • A∪(B∩C)=(A∪B)∩(A∪C)A \cup (B \cap C) = (A \cup B) \cap (A \cup C)A∪(B∩C)=(A∪B)∩(A∪C)
    • A∩(B∪C)=(A∩B)∪(A∩C)A \cap (B \cup C) = (A \cap B) \cup (A \cap C)A∩(B∪C)=(A∩B)∪(A∩C)

  • 德摩根律：

    • A∪B‾=A‾∩B‾\overline{A \cup B} = \overline{A} \cap \overline{B}A∪B=A∩B
    • A∩B‾=A‾∪B‾\overline{A \cap B} = \overline{A} \cup \overline{B}A∩B=A∪B

2. 逻辑与命题

2.1 命题逻辑

  • 命题连接词：

    • 否定：¬P\neg P¬P
    • 合取：P∧QP \land QP∧Q
    • 析取：P∨QP \lor QP∨Q
    • 蕴含：P→QP \rightarrow QP→Q
    • 等价：P↔QP \leftrightarrow QP↔Q

  • 逻辑等价：

    • P→Q≡¬P∨QP \rightarrow Q \equiv \neg P \lor QP→Q≡¬P∨Q
    • P↔Q≡(P→Q)∧(Q→P)P \leftrightarrow Q \equiv (P \rightarrow Q) \land (Q \rightarrow P)P↔Q≡(P→Q)∧(Q→P)

  • 德摩根律：

    • ¬(P∧Q)≡¬P∨¬Q\neg (P \land Q) \equiv \neg P \lor \neg Q¬(P∧Q)≡¬P∨¬Q
    • ¬(P∨Q)≡¬P∧¬Q\neg (P \lor Q) \equiv \neg P \land \neg Q¬(P∨Q)≡¬P∧¬Q

2.2 谓词逻辑

  • 全称量词：

    • ∀x P(x)\forall x \, P(x)∀xP(x)

  • 存在量词：

    • ∃x P(x)\exists x \, P(x)∃xP(x)

  • 量词否定：

    • ¬∀x P(x)≡∃x ¬P(x)\neg \forall x \, P(x) \equiv \exists x \, \neg P(x)¬∀xP(x)≡∃x¬P(x)
    • ¬∃x P(x)≡∀x ¬P(x)\neg \exists x \, P(x) \equiv \forall x \, \neg P(x)¬∃xP(x)≡∀x¬P(x)

3. 关系与函数

3.1 关系

  • 关系表示：

    • 有序对：(a,b)∈R(a, b) \in R(a,b)∈R

  • 关系性质：

    • 自反性：∀a∈A,(a,a)∈R\forall a \in A, (a, a) \in R∀a∈A,(a,a)∈R
    • 对称性：∀a,b∈A,(a,b)∈R⇒(b,a)∈R\forall a, b \in A, (a, b) \in R \Rightarrow (b, a) \in R∀a,b∈A,(a,b)∈R⇒(b,a)∈R
    • 传递性：∀a,b,c∈A,(a,b)∈R∧(b,c)∈R⇒(a,c)∈R\forall a, b, c \in A, (a, b) \in R \land (b, c) \in R \Rightarrow (a, c) \in R∀a,b,c∈A,(a,b)∈R∧(b,c)∈R⇒(a,c)∈R

  • 闭包：

    • 自反闭包：R∪{(a,a)∣a∈A}R \cup \{(a, a) \mid a \in A\}R∪{(a,a)∣a∈A}
    • 对称闭包：R∪R−1R \cup R^{-1}R∪R−1
    • 传递闭包：R+=⋃n=1∞RnR^+ = \bigcup_{n=1}^{\infty} R^nR+=⋃n=1∞​Rn

3.2 函数

  • 函数定义：

    • f:A→Bf: A \rightarrow Bf:A→B

  • 函数性质：

    • 单射：∀a1,a2∈A,f(a1)=f(a2)⇒a1=a2\forall a_1, a_2 \in A, f(a_1) = f(a_2) \Rightarrow a_1 = a_2∀a1​,a2​∈A,f(a1​)=f(a2​)⇒a1​=a2​
    • 满射：∀b∈B,∃a∈A,f(a)=b\forall b \in B, \exists a \in A, f(a) = b∀b∈B,∃a∈A,f(a)=b
    • 双射：既是单射又是满射。

4. 图论

4.1 基本概念

  • 图表示：

    • 无向图：G=(V,E)G = (V, E)G=(V,E)
    • 有向图：G=(V,E)G = (V, E)G=(V,E)

  • 度：

    • 无向图：deg⁡(v)\deg(v)deg(v)
    • 有向图：deg⁡+(v)\deg^+(v)deg+(v)（出度），deg⁡−(v)\deg^-(v)deg−(v)（入度）

4.2 图的连通性

  • 连通图：

    • 无向图：任意两点间存在路径。
    • 有向图：强连通（任意两点间双向路径）。

  • 欧拉回路：

    • 无向图：所有顶点的度数为偶数。
    • 有向图：所有顶点的入度等于出度。

  • 哈密顿回路：

    • 存在一条经过每个顶点一次的回路。

5. 代数结构

5.1 群

  • 群定义：
    • 封闭性：∀a,b∈G,a⋅b∈G\forall a, b \in G, a \cdot b \in G∀a,b∈G,a⋅b∈G
    • 结合律：∀a,b,c∈G,(a⋅b)⋅c=a⋅(b⋅c)\forall a, b, c \in G, (a \cdot b) \cdot c = a \cdot (b \cdot c)∀a,b,c∈G,(a⋅b)⋅c=a⋅(b⋅c)
    • 单位元：∃e∈G,∀a∈G,a⋅e=e⋅a=a\exists e \in G, \forall a \in G, a \cdot e = e \cdot a = a∃e∈G,∀a∈G,a⋅e=e⋅a=a
    • 逆元：∀a∈G,∃a−1∈G,a⋅a−1=a−1⋅a=e\forall a \in G, \exists a^{-1} \in G, a \cdot a^{-1} = a^{-1} \cdot a = e∀a∈G,∃a−1∈G,a⋅a−1=a−1⋅a=e

5.2 环与域

  • 环：

    • 加法群，乘法半群，分配律成立。

  • 域：

    • 加法群，乘法群（除去零元），分配律成立。

  • 组合：
